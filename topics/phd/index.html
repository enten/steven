    Android 5.1.1 x86_64

    15 Jun, 2015 · Read in about 5 Min (1100 Words) · Comments
    android lollipop virtualization qemu embedded system amd64

    Introduction Le projet android-x86 est un fork d’Android Open Source Project (AOSP) permettant d’installer Android sur PC (la compilation de ses sources produit une image CD bootable). Ce fork fût longtemps la seule solution pour compiler simplement Android pour plateforme x86. Avec Android 5 Lollipop, Google a ouvert la voix à la compilation d’AOSP pour architectures 64-bits (ARM et x86). Cette possibilité ne rend pas le fork android-x86 obsolète pour autant.
